# Diário de Pesquisa MEXT

Este é o meu repositório pessoal de acompanhamento de estudos e pesquisa para a bolsa de estudos do governo japonês (MEXT). O objetivo deste projeto é documentar a evolução da minha pesquisa sobre estratégias pedagógicas inclusivas para o ensino de língua japonesa, com foco em adaptação de materiais para alunos no espectro autista (TEA).

## 🚀 Sobre o Projeto
Este diário foi desenvolvido para organizar minhas leituras, descobertas e a aplicação de métodos de estudo estruturados, como o método TEACCH, voltados à aquisição de ideogramas (Kanji).

## 🛠️ Tecnologias Utilizadas
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Banco de Dados:** Supabase (PostgreSQL)
- **Hospedagem:** GitHub Pages

## 📂 Estrutura do Projeto
- `index.html`: Página principal com objetivos e metodologia.
- `diario.html`: Visualização dos registros de estudos.
- `admin.html`: Painel administrativo para inserção de novos conteúdos.
- `style.css`: Estilização e identidade visual do projeto.

## 📈 Acompanhamento
Através deste repositório, mantenho a transparência e a organização necessárias para o rigor acadêmico exigido na jornada de pesquisa.
